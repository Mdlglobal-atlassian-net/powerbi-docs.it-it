# <a name="create-with-power-bi"></a>Creare con Power BI

L'utente è un autore di Power BI? Si effettuano connessioni ai dati, si progettano report e si creano dashboard?  L'utente è uno sviluppatore che crea le proprie app? Si gestisce Power BI? Queste risorse consentono di accelerare le attività. Benvenuti.

L'utente è più di un semplice consumer di Power BI? Di nuovo benvenuti. È consigliabile iniziare con [Power BI per i consumer](consumer/power-bi-consumer-overview.md).
