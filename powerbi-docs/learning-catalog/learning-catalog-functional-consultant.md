# <a name="functional-application-consultant-learning-catalog"></a>Catalogo delle risorse di apprendimento per consulenti di applicazioni funzionali

Si è esperti dell'implementazione per un dominio aziendale? 

Nel catalogo seguente le conoscenze sono organizzate da quelle di base ai domini specifici e dalle più semplici alle più avanzate. Se il contenuto è disponibile in più formati, verrà comunicato in modo da consentire all'utente di scegliere il formato di training più adatto alle proprie esigenze.

Per i consulenti funzionali, la ricerca dell'analisi processo-attività ha dimostrato che tutti devono comprendere il set di contenuti di base e un'area di dominio.
