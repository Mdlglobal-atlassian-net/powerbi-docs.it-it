# <a name="copy-and-paste-a-visualization-in-a-report"></a>Copiare e incollare una visualizzazione in un report

Gli oggetti visivi in Power BI possono essere copiati da una pagina del report nella stessa pagina o in una pagina diversa dello stesso report. 

Per copiare e incollare una visualizzazione sono necessarie autorizzazioni di modifica per il report. Nel servizio Power BI Ã¨ necessario aprire il report in [Visualizzazione di modifica](../consumer/end-user-reading-view.md).

## <a name="copy-and-paste-a-visual"></a>Copiare e incollare un oggetto visivo

1. Aprire un report che include almeno una visualizzazione.  

2. Selezionare la visualizzazione e usare **CTRL+C** per copiare e **CTRL+V** per incollare.
