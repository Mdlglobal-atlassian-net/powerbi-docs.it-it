## <a name="power-bi-premium"></a>Power BI Premium
**Che cos'è Power BI Premium?**  
Power BI Premium è una nuova offerta basata sulla capacità che include:

* La flessibilità di pubblicare i report su vasta scala nell'organizzazione, senza richiedere ai destinatari di avere una licenza per ogni utente.
* Scalabilità e prestazioni maggiori della capacità dedicata nel servizio Power BI.
* La possibilità di gestire asset BI in locale con il server di report di Power BI.
* Una sola superficie API, un set coerente di funzionalità e l'accesso alle funzionalità più recenti per l'analisi incorporata.

**Che cosa fa Power BI Premium? Come funziona?**  
Power BI Premium rappresenta la capacità nel servizio Power BI allocata esclusivamente a ogni organizzazione ed è supportata da hardware dedicato, completamente gestito da Microsoft. Le organizzazioni possono scegliere di applicare la loro capacità dedicata su vasta scala oppure di allocarla ad aree di lavoro assegnate in base al numero di utenti, alle esigenze del carico di lavoro o ad altri fattori e quindi aumentarla o ridurla quando cambiano i requisiti.

**In che modo Power BI Pro differisce da Power BI Premium? Perché Power BI Premium è necessario?**  
Power BI Premium è una licenza basata sulla capacità, mentre Power BI Pro è una licenza basata sull'utente.

Power BI Premium rappresenta la capacità nel servizio Power BI allocata esclusivamente a ogni organizzazione ed è supportata da hardware dedicato, completamente gestito da Microsoft, che rimuove le limitazioni sulle dimensioni dei set di dati e sulle frequenze di aggiornamento e supporta l'isolamento dovuto all'hardware dedicato. Power BI Premium fornisce anche la flessibilità di pubblicare i report su vasta scala nell'organizzazione, senza richiedere ai destinatari di avere una licenza per ogni utente.

Power BI Pro è destinato agli utenti che pubblicano report, condividono dashboard, collaborano con i colleghi nelle aree di lavoro e partecipano ad altre attività correlate, quali:

* Modificare e salvare visualizzazioni personalizzate
* Creare dashboard personali (aggiungere a nuovo dashboard)
* Analizzare dati in Excel o Power BI Desktop
* Condividere con il supporto dell'applicazione Excel Web
* Condividere dashboard e collaborare con i gruppi di Office 365
* Integrare contenuti con Microsoft Teams

**È necessario Power BI Pro per usare Power BI Premium?**  
Sì. Power BI Pro è obbligatorio per pubblicare report, condividere dashboard, collaborare con i colleghi nelle aree di lavoro e partecipare ad altre attività correlate.

**Quanto costa Power BI Premium? Quante SKU saranno rese disponibili?**  
È possibile acquistare Power BI Premium in base al numero di memorie centrali virtuali. I prezzi sono disponibili nella [pagina Prezzi di Power BI](https://powerbi.microsoft.com/pricing/). Per altre informazioni su nodi e v-core, vedere il [white paper di Microsoft Power BI Premium](https://aka.ms/pbipremiumwhitepaper). Usare anche questa [calcolatrice](https://powerbi.microsoft.com/calculator/) per ottenere una stima della capacità di Premium che potrebbe essere necessaria.

**Cosa si intende per "capacità"?**  
"Capacità" fa riferimento a una capacità denominata di cui un amministratore esegue il provisioning attraverso il portale di amministrazione capacità di Power BI Premium. La capacità è un raggruppamento di uno o più nodi. Per altre informazioni, vedere il [white paper di Microsoft Power BI Premium](https://aka.ms/pbipremiumwhitepaper).

**Come viene fatturato Power BI Premium?**  
Power BI Premium viene fatturato mensilmente con un impegno annuo.

**Come sarà possibile acquistare Power BI Premium?**  
Power BI Premium è disponibile come componente aggiuntivo di Power BI Pro nell'interfaccia di amministrazione di Office 365. Per altre informazioni, vedere [How to purchase Power BI Premium](service-admin-premium-purchase.md) (Come acquistare Power BI Premium). È anche possibile contattare il rappresentante di vendita Microsoft per altre informazioni.

**Come sapere quanta capacità di Power BI Premium è necessario acquistare?**  
Usare questa [calcolatrice](https://powerbi.microsoft.com/calculator/) dei costi per ottenere una stima della quantità di capacità di Power BI Premium che potrebbe essere necessaria.

**Quando sarà disponibile Power BI Premium? Sarà disponibile contemporaneamente in tutte le aree geografiche?**  
Power BI Premium sarà disponibile a livello generale verso la fine del secondo trimestre del 2017 per tutte le aree geografiche attualmente supportate dal servizio Power BI.

**Power BI Premium soddisferà la conformità normativa e di certificazione al momento del rilascio? E per quanto riguarda i cloud nazionali?**  
Power BI Premium sarà disponibile a livello generale verso la fine del secondo trimestre del 2017 per tutte le aree geografiche attualmente supportate dal servizio Power BI. La disponibilità dei cloud nazionali, in particolare in Germania, Cina e Governo degli Stati Uniti, è progettata per il terzo trimestre del 2017. Gli standard di conformità normativa e di certificazione si applicheranno a Power BI Premium al momento della disponibilità generale.

**Power BI Premium è disponibile con Office 365 E5?**  
Power BI Premium è disponibile come componente aggiuntivo di Power BI Pro. Office 365 E5 include Power BI Pro. I clienti di E5 possono acquistare Premium come componente aggiuntivo alle licenze Pro esistenti.

**Che cos'è il server di report di Power BI? Ciò significa che Power BI sarà disponibile in locale?**  
Il server di report di Power BI è disponibile come parte di Power BI Premium ed è compatibile con i report di SQL Server Reporting Services, consentendo la distribuzione dei report interattivi di Power BI, nonché dei report impaginati, in locale all'interno del firewall dell'organizzazione. Le organizzazioni possono scegliere Power BI nel cloud oppure decidere di mantenere i report in locale con il server di report Power BI, spostandoli nel cloud secondo i propri ritmi.

Le funzionalità di Power BI, come i dashboard in tempo reale, le query in linguaggio naturale e altre nel servizio Power BI non sono disponibili come parte del server di report di Power BI. 

Per altre informazioni sul server di report di Power BI, vedere [Introduzione al server di report di Power BI](report-server/get-started.md).

**Il server di report di Power BI è la funzionalità resa disponibile da Microsoft come anteprima tecnica dei Report di Power BI per SSRS? Perché ora fa parte di Power BI quando prima era una funzionalità di SQL?**  
Il server di report di Power BI è disponibile come parte di Power BI Premium ed è compatibile con i report di SQL Server Reporting Services, consentendo la distribuzione dei report interattivi di Power BI, nonché dei report impaginati, in locale all'interno del firewall dell'organizzazione.

Annunciando Power BI Premium, Microsoft rivela per la prima volta il modo in cui questa funzionalità verrà portata in commercio.

Per altre informazioni sul server di report di Power BI, vedere [Introduzione al server di report di Power BI](report-server/get-started.md).

**Cosa sono le app di Power BI?**  
Stiamo trasformando i pacchetti di contenuto in app di Power BI per migliorare il modo in cui gli utenti scoprono ed esplorano le informazioni dettagliate su scala aziendale. Disponibili subito, le app di Power BI offrono un modo semplificato di distribuire dashboard e report a specifici utenti, gruppi o a un'intera organizzazione. Gli utenti aziendali possono facilmente installare queste app e spostarsi con facilità, centralizzando il contenuto in un'unica posizione ed eseguendo l'aggiornamento automatico. 

Per altre informazioni sulle app, vedere [Cosa sono le app in Power BI?](service-install-use-apps.md).

**Si vogliono apportare modifiche a Power BI Pro, Power BI gratuito, Power BI Desktop e ad altri come parte dell'implementazione di Power BI Premium?**  
Oggi verrà semplificata anche la distinzione tra Power BI Pro e il servizio gratuito. Abbiamo ricevuto commenti e suggerimenti sul fatto che, nonostante il servizio gratuito sia destinato all'uso personale e Power BI Pro consenta la collaborazione, le differenze funzionali tra le due versioni hanno creato confusione tra gli utenti. In futuro, l'obiettivo sarà di migliorare il servizio gratuito per avere la stessa funzionalità di Power BI Pro, ma anche di limitare le funzionalità di condivisione e collaborazione solo agli utenti di Power BI Pro.

Gli utenti del servizio Power BI gratuito otterranno l'accesso a tutte le origini dati, limiti maggiori per l'archiviazione nell'area di lavoro e una maggiore frequenza di aggiornamento e streaming. Queste modifiche diventeranno effettive a partire dal 1 giugno 2017. Altre informazioni sono disponibili in [Iscrizione a Power BI in modalità self-service](service-self-service-signup-for-power-bi.md). Power BI Desktop resta disponibile gratuitamente.

Per informazioni relative all'offerta della versione di valutazione Pro estesa, vedere [Extended Pro Trial activation](service-extended-pro-trial.md) (Attivazione della versione di valutazione Pro estesa).

## <a name="power-bi-free"></a>Power BI gratuito
**Quali funzionalità verranno aggiunte al servizio gratuito?**  
A partire dal 1 giugno 2017, il servizio gratuito avrà funzionalità equivalenti a Power BI Pro, ad eccezione della condivisione, della distribuzione aziendale e della collaborazione. Ciò include lo stesso limite dell'area di lavoro di 10 GB, fino a 8 aggiornamenti pianificati giornalieri per i set di dati e una velocità dati di streaming massima di 1M/ora. Verrà incluso anche l'accesso a tutte le origini dati, incluse quelle disponibili attraverso il gateway dati locale.

**Quali funzionalità di Power BI Pro non sono disponibili con il servizio gratuito?**  
La condivisione del dashboard peer-to-peer, le aree di lavoro per le app (in precedenza denominate aree di lavoro di gruppo), le app Analizza in Excel e Power BI (a meno che non siano supportate da una capacità Premium) sono funzionalità limitate a Power BI Pro.

**Quando diventeranno effettive queste modifiche?**  
Le modifiche al servizio gratuito entreranno in vigore il 1 giugno 2017.

**In che modo gli utenti del servizio gratuito interessati saranno avvisati di queste modifiche?**  
Gli utenti attivi del servizio gratuito interessati dalle modifiche apportate il 1 giugno 2017 riceveranno una comunicazione via posta elettronica da parte di Microsoft, oltre alle notifiche quando effettuano l'accesso al servizio Power BI a partire dal 3 maggio 2017.

Gli utenti esistenti del servizio gratuito che sono stati attivi nell'ultimo anno potranno provare una versione di valutazione gratuita estesa di 12 mesi di Power BI Pro. L'offerta comprende le funzionalità complete di Power BI Pro per garantire che gli utenti abbiano tutto il tempo di adattarsi all'uso del servizio. Per informazioni relative all'offerta della versione di valutazione Pro estesa, vedere [Extended Pro Trial activation](service-extended-pro-trial.md) (Attivazione della versione di valutazione Pro estesa).

**Gli utenti perderanno l'accesso al contenuto (dashboard, report)?**  
No. Non perderanno l'accesso a nessun contenuto caricato nel servizio Power BI. Il contenuto condiviso con altri utenti potrebbe non essere più accessibile dai destinatari dal 1 giugno 2017. Allo stesso modo, da quale momento potrebbero non avere più accesso al contenuto che altri utenti hanno condiviso con loro.

Gli utenti esistenti del servizio gratuito che sono stati attivi nell'ultimo anno potranno provare una versione di valutazione gratuita estesa di 12 mesi di Power BI Pro. L'offerta comprende le funzionalità complete di Power BI Pro per garantire che gli utenti abbiano tutto il tempo di adattarsi all'uso del servizio. Per informazioni relative all'offerta della versione di valutazione Pro estesa, vedere [Extended Pro Trial activation](service-extended-pro-trial.md) (Attivazione della versione di valutazione Pro estesa).

**In che modo gli utenti sapranno di essere idonei per la versione di valutazione estesa di Power BI Pro?**  
Consultare le [condizioni per l’utilizzo](https://aka.ms/power-bi-trial) della versione di valutazione estesa di Power BI Pro.

Per informazioni relative all'offerta della versione di valutazione Pro estesa, vedere [Extended Pro Trial activation](service-extended-pro-trial.md) (Attivazione della versione di valutazione Pro estesa).

**Cosa dovranno fare gli utenti se non saranno idonei per la versione di valutazione estesa di Power BI Pro?**  
Iscriversi per ricevere una versione di valutazione standard di Power BI Pro per 60 giorni [qui](https://app.powerbi.com/signupredirect?pbi_source=web).

**L'introduzione di Power BI Premium modifica le funzionalità del servizio gratuito?**  
Gli utenti del servizio gratuito con accesso alle capacità dedicata in Power BI Premium avranno la possibilità di ricevere contenuto distribuito dagli utenti di Power BI Pro attraverso le app pubblicate.

## <a name="power-bi-embedded"></a>Power BI Embedded
**Cosa cambia in Power BI Embedded?**  
Come parte di Power BI Premium, stiamo unendo Power BI Embedded con il servizio Power BI per fornire un'unica superficie API, un set di funzionalità coerente e l'accesso alle ultime funzionalità. In futuro si invitano gli utenti interessati a incorporare Power BI nelle app di iniziare con Power BI Desktop e passare alla distribuzione con Power BI Premium.

**Quali sono i vantaggi di Power BI Premium per l'incorporamento dell'analisi nelle app?**  
Power BI Premium è un'offerta basata sulle capacità che viene eseguita su hardware dedicato ed è completamente gestita da Microsoft come parte del servizio Power BI. Tenendo presente il ciclo di vita di sviluppo dell'applicazione, Power BI Premium offre nuovi vantaggi quali costi flessibili e prevedibili, una migliore gestione del contenuto per gli sviluppatori e la disponibilità completa delle funzionalità di Power BI per gli scenari incorporati.

**È necessario eseguire la migrazione dell'applicazione esistente basata su Power BI Embedded?**  
Power BI Embedded continuerà a essere disponibile per le app esistenti secondo la pianificazione seguente:

| Contratto di licenza | Disponibilità di Power BI Embedded |
| --- | --- |
| Contratto Enterprise |Fino alla scadenza del contratto esistente |
| Diretto e CSP |Un anno dalla disponibilità generale di Power BI Premium |

Sì, è necessario avviare alcune riflessioni sulla migrazione della soluzione analitica incorporata a Power BI Premium. Per altre informazioni, vedere [Come eseguire la migrazione del contenuto della raccolta di aree di lavoro di Power BI Embedded in Power BI](developer/migrate-from-powerbi-embedded.md).

**Come eseguire la migrazione dell'applicazione esistente basata su Power BI Embedded?**  
Per istruzioni sulla migrazione delle app, consultare la [documentazione sulla migrazione](developer/migrate-from-powerbi-embedded.md).

**Microsoft sta riconsiderando il suo impegno per il rilascio di una versione gratuita di Power BI Embedded?**  
Nel modello precedente, era disponibile un numero limitato di sessioni per lo sviluppo e i test. Nel nuovo scenario, i clienti continueranno ad avere la possibilità di avviare lo sviluppo e i test gratuitamente, in modo essere pronti per la produzione. I clienti trarranno vantaggio da una versione di valutazione di Power BI Pro per 2 mesi per iniziare le fasi di sviluppo e test. Power BI Desktop, l'esplorazione dei dati e lo strumento di creazione report continueranno a essere disponibili gratuitamente.

**Power BI Embedded è ancora un servizio di Azure? Dove acquistare, eseguire il provisioning e gestire Power BI Embedded?**  
Power BI Embedded verrà unito al servizio Power BI e di conseguenza sarà possibile acquistarlo, eseguirne il provisioning e gestirlo come tale in futuro. Per altre informazioni, vedere la [Incorporamento con Power BI](developer/embedding.md).

## <a name="power-bi-report-server"></a>Server di report Power BI
**Perché il server di report di Power BI è incluso in Power BI Premium invece di essere venduto come prodotto distinto?**  
Power BI Premium introduce la possibilità di gestire asset BI in locale con il server di report di Power BI. Il server di report di Power BI è un server locale che consente la distribuzione dei report interattivi di Power BI, nonché dei report impaginati, completamente all'interno del firewall dell'organizzazione. Con Power BI Premium lo stesso numero di memorie centrali virtuali di cui un'organizzazione esegue il provisioning nel cloud può anche essere distribuito in locale attraverso il server di report di Power BI, senza la necessità di dividere la capacità. Le organizzazioni possono scegliere Power BI nel cloud oppure decidere di mantenere i report in locale con il server di report Power BI, spostandoli nel cloud secondo i propri ritmi.

Per altre informazioni vedere [Introduzione al server di report di Power BI](report-server/get-started.md).

**In che modo i clienti possono ottenere l'accesso al server di report di Power BI?**  
Il server di report di Power BI è disponibile attraverso Power BI Premium o come vantaggio per i clienti con SQL Server EE con SA. Contattare il rappresentante di vendita Microsoft per altre informazioni.

**Per quale motivo il server di report di Power BI non è disponibile come vantaggio per i clienti di SQL Server Standard o Enterprise Edition?**  
Il server di report di Power BI è disponibile per i clienti di SQL Server con licenze EE + SA o per l'acquisto attraverso Power BI Premium. Power BI è un servizio SaaS e il server di report di Power BI seguirà lo stesso modello di distribuzione basato su sottoscrizione.
